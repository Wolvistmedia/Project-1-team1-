Azure Certification Tracks

Choose your certification path based on your role and career goals. Each track builds expertise in core Azure competencies — from cloud basics to advanced architecture and security.

Track 1: Fundamental Level — Cloud Basics

Entry-level certifications ideal for beginners, sales, management, and anyone starting their cloud journey.

1. AZ-900: Microsoft Azure Fundamentals

Beginner | Cloud Concepts | 1 month

Level: Beginner | Focus: Cloud concepts, Azure services, pricing, governance

The perfect starting point for anyone new to Azure. Learn core cloud concepts, Azure services overview, pricing models, and governance fundamentals. Ideal for all roles — technical and non-technical alike.

Intermediate certifications for cloud engineers, security engineers, developers, and DevOps professionals.

1. AZ-104: Microsoft Azure Administrator Associate

Intermediate | Cloud Administration | 2–3 months

Level: Intermediate | Focus: Identity, compute, storage, networking, monitoring

Master Azure administration skills — managing subscriptions, identities, virtual networks, storage accounts, virtual machines, and monitoring solutions. The core certification for Azure administrators.
