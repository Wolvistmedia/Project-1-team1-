Microsoft Azure Solutions Architect AZ-305

Design end-to-end Azure solutions for enterprise requirements — governance, identity, compute, networking, data, and business continuity. The expert-level certification for cloud architects building production Azure environments.

Exam: AZ-305 Level: Expert Duration: 6–8 Weeks
Start Training Download Syllabus Book Demo
✓ Expert-Level Certification ✓ Architecture Design Labs ✓ Prerequisite: AZ-104

Program Overview

Full AZ-305 exam objective coverage

Identity, governance, and compliance architecture

Compute, container, and infrastructure design

Network topology and connectivity design

Data storage and integration solutions

Business continuity and disaster recovery planning

Who Should Attend

Senior cloud engineers, Azure administrators targeting architect roles, solution designers, and IT professionals with hands-on Azure experience seeking the Expert-level certification.

Prerequisites

AZ-104 (Azure Administrator Associate) certification is required before appearing for AZ-305. Solid hands-on experience with Azure services, networking, identity, and storage is strongly recommended.

What You Get

Course materials, hands-on Azure architecture labs, practice exams, case study walkthroughs, instructor support, mock interviews, and a path toward AZ-400 DevOps Engineer Expert.

Course Curriculum

Comprehensive training covering all four AZ-305 exam domains — identity, compute, networking, data, and business continuity — with real-world architecture design scenarios.

Module 1 • 25–30%

Design Identity, Governance, and Monitoring Solutions

Architect identity systems, governance frameworks, and monitoring strategies for enterprise Azure environments.

Identity and Access Architecture

  • Design Azure AD (Entra ID) tenant architecture
  • Design hybrid identity with Azure AD Connect
  • Architect external identity and B2B/B2C solutions
  • Design Privileged Identity Management (PIM) strategy

Governance and Compliance Design

  • Design management group and subscription hierarchy
  • Architect Azure Policy and initiative assignments
  • Design resource tagging and naming conventions
  • Plan Azure Blueprints for landing zone governance

Monitoring and Logging Architecture

  • Design Azure Monitor and Log Analytics strategy
  • Architect alerts, dashboards, and workbook solutions
  • Design diagnostic settings and data collection rules
  • Integrate with Microsoft Sentinel for security monitoring

Module 2 • 25–30%

Design Data Storage Solutions

Architect scalable, secure, and performant data storage and integration solutions in Azure.

Relational Data Solutions

  • Design Azure SQL Database and Managed Instance
  • Architect Azure SQL Elastic Pools for multi-tenant apps
  • Design Azure Database for MySQL and PostgreSQL
  • Plan high availability and geo-replication strategies

Non-Relational and Storage Design

  • Design Azure Cosmos DB for globally distributed apps
  • Architect Azure Storage (Blob, Files, Tables, Queues)
  • Design storage tiers, lifecycle policies, and replication
  • Architect Azure Data Lake Storage Gen2 solutions

Data Integration Solutions

  • Design Azure Data Factory pipelines and workflows
  • Architect Azure Synapse Analytics workspaces
  • Design event-driven data flows with Event Hub
  • Integrate Azure Stream Analytics for real-time processing

Module 3 • 25–30%

Design Infrastructure Solutions

Design compute architecture, container platforms, application delivery, and network topology for enterprise workloads.

Compute and Application Architecture

  • Design VM-based compute with availability sets and zones
  • Architect Azure App Service and Azure Functions solutions
  • Design auto-scaling strategies for workloads
  • Architect Azure Batch for HPC and parallel processing

Container and Microservices Design

  • Design Azure Kubernetes Service (AKS) architectures
  • Architect Azure Container Apps and microservices
  • Design Azure Container Registry (ACR) for image management
  • Plan service mesh and ingress controller strategies

Network Topology and Connectivity

  • Design hub-and-spoke and Virtual WAN topologies
  • Architect hybrid connectivity (VPN Gateway, ExpressRoute)
  • Design Azure Firewall, NSGs, and WAF placement
  • Architect Azure Front Door and Traffic Manager solutions

Module 4 • 10–15%

Design Business Continuity Solutions

Architect high availability, backup, and disaster recovery strategies to meet enterprise SLA and RTO/RPO requirements.

High Availability Design

  • Design multi-region active-active architectures
  • Architect availability zones and region pairs
  • Design Azure Load Balancer and Application Gateway HA
  • Plan SLA targets and uptime commitments

Backup and Disaster Recovery

  • Architect Azure Backup for VMs, SQL, and file shares
  • Design Azure Site Recovery (ASR) replication topology
  • Plan RTO/RPO requirements and failover testing
  • Design Recovery Services Vault and backup policies
