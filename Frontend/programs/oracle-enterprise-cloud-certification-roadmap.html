Certification Roadmap Courses

1) Oracle Database 19c: Administration Workshop

Core administration tasks including installation, configuration, and user management.

Focus: Build a solid Oracle DBA foundation for production database operations.

2) Oracle Cloud Infrastructure Architect Associate

Design secure and scalable cloud solutions on Oracle Cloud Infrastructure.

Focus: OCI architecture fundamentals and cloud design best practices.

3) Oracle WebLogic Server 14c: Administration I

WebLogic domain setup, server configuration, and basic administration tasks.

Focus: Start administering enterprise Java middleware environments.

4) Oracle Data Integrator 12c: Integration and Administration Ed 3

Data integration mappings, workflows, and ODI administration essentials.

Focus: Build ETL-style integration capabilities with ODI.

5) Oracle Database 19c: Multitenant Architecture

Container and pluggable database architecture for consolidation and agility.

Focus: Multitenant design and administration strategy for modern Oracle deployments.

6) Oracle Database 19c: SQL Workshop

SQL querying, joins, functions, data manipulation, and analytical queries.

Focus: Strong SQL proficiency for administration, reporting, and development support.

7) Oracle Exadata Database Machine: Implementation and Administration

Deploy and administer Exadata for high-performance database workloads.

Focus: Engineered systems operations and optimization practices.

8) Oracle WebLogic Server 14c: Administration II

Advanced WebLogic security, clustering, troubleshooting, and tuning.

Focus: Deepen middleware administration for enterprise-scale environments.

9) Application Integration on Oracle Cloud Ed 4

Design and build integrations across SaaS and on-prem systems using Oracle Cloud services.

Focus: Cloud integration architecture and orchestration workflows.

10) Oracle Database 23ai: New Features for Administrators Live Class

Learn new administration capabilities, automation, and feature updates in Oracle 23ai.

Focus: Upgrade readiness and modernization for current Oracle estates.

11) Oracle SOA Suite 12c: Build Composite Applications Ed 2

Develop composite services, orchestrations, and reusable integration components.

Focus: Service-oriented integration design with Oracle SOA Suite.

12) Primavera P6 Professional Fundamentals Rel 19 Ed 1

Project creation, scheduling basics, resources, and baseline planning.

Focus: Foundation-level enterprise project planning and controls.

13) Oracle Database 19c: Data Guard Administration Workshop

Configure and manage Data Guard for disaster recovery and high availability.

Focus: Business continuity and resilient database architecture operations.

14) Primavera P6 Professional Project Management Rel 19 Ed 1

Advanced project controls, tracking, optimization, and portfolio execution workflows.

Focus: End-to-end project management execution using Primavera P6.

15) Oracle Database 19c: Backup and Recovery

RMAN, restore workflows, recovery scenarios, and data protection best practices.

Focus: Reliable backup and recovery operations for mission-critical databases.
