Professional Track Modules

Module 1: CompTIA Network+ Foundations

Entry-level networking concepts for beginners and helpdesk/NOC roles.

Networking Basics

Network types, topologies, and models • OSI and TCP/IP fundamentals • Common network devices and roles • Addressing and subnetting basics

Core Protocols

IPv4, IPv6, DNS, DHCP, and NAT • TCP vs UDP behavior • Ports, sessions, and packet flow • Protocol troubleshooting basics

Beginner Job Readiness

Helpdesk and NOC workflow • Ticket handling and escalation • Basic monitoring tools usage • Network documentation standards

Module 2: Network+ Troubleshooting, Wireless, and Security

Operational support skills from your requested Network+ topic set.

Troubleshooting Methods

Structured troubleshooting approach • Layer-by-layer fault isolation • Connectivity and latency diagnosis • CLI and analyzer basics

Wireless Networking

Wi-Fi standards and channels • Access point and controller concepts • Coverage, roaming, and interference • Wireless troubleshooting checks

Network Security Basics

Firewalls, ACLs, and segmentation • Authentication and access controls • Threat types in enterprise networks • Secure configuration baseline

Module 3: Cisco CCNA Core Networking

Routing and switching fundamentals aligned with CCNA outcomes.

Routing and Switching

Switching logic and MAC learning • Inter-VLAN routing fundamentals • Static and dynamic routing • STP and redundancy concepts

VLAN Configuration

VLAN planning and segmentation • Trunks and native VLAN • Access/trunk port troubleshooting • Routing between VLANs

IP Connectivity

Addressing strategy and design • Gateway and route validation • Reachability testing techniques • Connectivity incident handling

Module 4: CCNA Security and Automation

CCNA-level network security basics and automation fundamentals.

Security Basics

Switch port and access hardening • Basic ACL and filtering use cases • Identity and device trust concepts • Secure network operations

Network Automation

Automation overview for networks • Controller-based networking concepts • API-driven configuration basics • Change consistency and rollback

CCNA Role Alignment

Network engineer responsibilities • SOC analyst networking expectations • Cybersecurity team collaboration • Interview-oriented scenario drills

Module 5: Cisco CCNP Enterprise Depth

Advanced routing, enterprise networking, and troubleshooting depth.

Advanced Routing

OSPF, BGP, and policy control • Route redistribution strategies • Traffic engineering concepts • High-availability routing design

Enterprise Networking

Campus and branch architectures • Core-distribution-access model • Scalable segmentation patterns • Operational resilience planning

Advanced Troubleshooting

Complex incident diagnosis • Control-plane and data-plane checks • Protocol instability handling • Post-incident optimization

Module 6: CCNP SD-WAN and Network Security

Enterprise WAN modernization and advanced security topics from CCNP path.

SD-WAN Architecture

Overlay-underlay fundamentals • Controller-based WAN operations • Application-aware routing • Branch policy orchestration

WAN Operations

Path selection and failover • QoS and performance management • Telemetry and health visibility • Troubleshooting SD-WAN flows

CCNP Security Integration

Segmentation and policy control • Secure branch and edge design • Threat-aware network design • Compliance-aligned operations

Module 7: Cisco CCIE Expert-Level Domains

Expert track for enterprise infrastructure, service provider, and data center networking.

Enterprise Infrastructure

Large-scale enterprise design • High availability at scale • Complex policy and segmentation • Optimization and resiliency tuning

Service Provider Networking

Provider-grade routing concepts • MPLS and traffic engineering overview • Carrier reliability patterns • Large backbone troubleshooting

Data Center Networking

Spine-leaf and fabric principles • East-west traffic design • Virtualization-aware networking • Data center security controls

Module 8: Juniper JNCIA and JNCIS Path

Juniper associate to specialist progression with routing and operations focus.

JNCIA Foundations

Juniper platform and OS basics • Routing protocol fundamentals • Basic switching and interfaces • Network operations workflow

JNCIS Specialist Focus

Enterprise routing and switching • Intermediate design decisions • Configuration validation • Production network tuning

Juniper Troubleshooting

Protocol and link issue diagnosis • Path and policy verification • Operational best practices • Escalation and root-cause reports

Module 9: Cloud Networking with Azure AZ-700

Designing and implementing Microsoft Azure networking solutions.

Virtual Networks

VNet and subnet architecture • Peering and segmentation • Private endpoints and isolation • Cloud routing strategy

Load Balancing and Gateways

Azure Load Balancer and App Gateway • VPN Gateway architecture • Traffic distribution patterns • Availability and failover design

Hybrid Networking

On-prem to cloud integration • ExpressRoute and VPN planning • Hybrid DNS and connectivity • Security and governance controls

Module 10: Security Networking Track (Security+ and CISSP)

Network security controls and architecture foundations for cybersecurity alignment.

CompTIA Security+ Networking

Network threat landscape • Defense-in-depth for networks • Secure access and authentication • Operational security monitoring

CISSP Security Architecture

Security design principles • Risk-based network controls • Governance and compliance mapping • Enterprise security operations

Career Outcome Mapping

Network security engineer pathway • SOC and NOC collaboration model • Incident readiness checklist • Certification strategy and timeline
