Who Should Attend

IT administrators, sysadmins moving to cloud, DevOps engineers adding Azure skills, and anyone targeting the AZ-104 certification or preparing for AZ-305 Azure Architect.

Prerequisites

Basic understanding of networking, storage, and operating systems. AZ-900 (Azure Fundamentals) recommended but not mandatory. No prior Azure experience required.

What You Get

Course materials, hands-on Azure lab access, practice exams, study guides, instructor support, mock interviews, and exam success tracking.

Course Curriculum

Comprehensive training covering all five AZ-104 exam domains with hands-on Azure labs and real-world scenarios.

Module 1 · 15–20%

Manage Azure Identities and Governance

Configure Azure Active Directory (Entra ID), RBAC, subscriptions, and governance policies.

Azure Active Directory (Entra ID)

  • • Create and manage users and groups
  • • Configure external identities and guest users
  • • Manage Azure AD joined and registered devices
  • • Configure self-service password reset (SSPR)

Role-Based Access Control (RBAC)

  • • Assign built-in Azure roles
  • • Create custom RBAC roles
  • • Manage role assignments at scope levels
  • • Implement least-privilege access principles

Subscriptions and Governance

  • • Configure management groups and subscriptions
  • • Apply Azure Policy and initiatives
  • • Implement resource tags and resource locks
  • • Configure cost management and budgets
Module 2 · 15–20%

Implement and Manage Storage

Create and secure Azure storage accounts, manage Blob, File, and hybrid storage solutions.

Storage Accounts

  • • Create and configure storage accounts
  • • Select redundancy options (LRS, ZRS, GRS, GZRS)
  • • Configure access tiers (Hot, Cool, Archive)
  • • Manage storage endpoints and firewalls

Blob and File Storage

  • • Configure Blob containers and access levels
  • • Implement Blob lifecycle management policies
  • • Deploy Azure Files and File Sync
  • • Create file shares and manage quotas

Storage Security

  • • Generate and manage Shared Access Signatures (SAS)
  • • Configure storage access keys and rotation
  • • Enable Azure AD-based storage authentication
  • • Configure stored access policies
Module 3 · 20–25%

Deploy and Manage Azure Compute Resources

Provision and manage virtual machines, containers, and application hosting services.

Azure Virtual Machines

  • • Deploy Windows and Linux VMs
  • • Configure VM sizes, disks, and images
  • • Manage VM availability sets and zones
  • • Configure VM Scale Sets (VMSS) for autoscaling

App Service and Containers

  • • Create and configure Azure App Service plans
  • • Deploy web apps and configure settings
  • • Run containerized apps with Azure Container Instances (ACI)
  • • Manage container images in Azure Container Registry (ACR)

Automation and Templates

  • • Deploy resources using ARM templates
  • • Automate tasks with Azure CLI and PowerShell
  • • Use Azure Automation runbooks
  • • Configure custom script extensions on VMs
Module 4 · 25–30%

Implement and Manage Virtual Networking

Design and manage Azure virtual networks, DNS, load balancing, and hybrid connectivity.

Virtual Networks and Subnets

  • • Create and configure Virtual Networks (VNet)
  • • Plan IP addressing schemes and subnets
  • • Implement VNet peering (same region and global)
  • • Configure service and private endpoints

Network Security

  • • Configure Network Security Groups (NSG) and rules
  • • Implement Application Security Groups (ASG)
  • • Deploy and configure Azure Firewall
  • • Configure Azure DDoS Protection

Load Balancing and DNS

  • • Deploy Azure Load Balancer (public and internal)
  • • Configure Application Gateway and WAF
  • • Set up Azure DNS zones and records
  • • Implement VPN Gateway and ExpressRoute basics
Module 5 · 10–15%

Monitor and Maintain Azure Resources

Configure monitoring, alerting, backup, and disaster recovery for Azure workloads.

Azure Monitor

  • • Configure diagnostic settings and activity logs
  • • Create metric alerts and action groups
  • • Set up Log Analytics workspaces
  • • Write basic KQL queries for insights

VM Monitoring and Insights

  • • Enable VM Insights and dependency maps
  • • Configure Azure Monitor Agent (AMA)
  • • Monitor VM performance and health
  • • Set up Network Watcher and flow logs

Backup and Recovery

  • • Configure Azure Backup for VMs and files
  • • Create and manage Recovery Services vaults
  • • Implement Azure Site Recovery (ASR) for DR
  • • Perform backup restores and failover tests
